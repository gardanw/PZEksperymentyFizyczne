{% endblock %} {% block teoria_display %}{% endblock %} {% block teoria %} Krzywa balistyczna – krzywa, po której poruszałby się wystrzelony punkt materialny bez napędu, uwzględniając działający na niego opór powietrza.

W praktyce krzywa balistyczna wyznacza tor lotu środka masy pocisku od punktu wylotu z lufy do punktu upadku. Kształt tego toru, tak jak w przypadku punktu materialnego, zależy od kąta nachylenia i prędkości początkowej pocisku oraz od wysokości, z której został on wystrzelony.

Krzywa balistyczna odpowiada w przybliżeniu paraboli, w rzeczywistości jest jednak nieco asymetryczna.

Ruch wystrzelonego, pod kątem Θ i z prędkością v0, pocisku można podzielić na dwie składowe, poziomą x oraz pionową y. Położenie pocisku w chwili t można obliczyć ze wzoru:
$$ { x(t)=x(t-\epsilon)+V_x(t-\epsilon)t } $$ $$ { y(t)=y(t-\epsilon)+V_y(t-\epsilon)t } $$ gdzie:
Vx(t) i Vy(t) to składowe prędkości, w chwili t dane wzorem: $$ { V_x(t)=V_x(t-\epsilon)+a_x(t-\epsilon)t } $$ $$ { V_y(t)=V_y(t-\epsilon)+a_y(t-\epsilon)t } $$ gdzie:
ax(t) i ay(t) to składowe przyśpieszenia, które w chwili t można obliczyć: $$ { a_x(t) = -\frac{1}{2}\alpha V_x(t) } $$ $$ { a_y(t) = -\frac{1}{2}\alpha V_y(t)-\frac{1}{2}g } $$ gdzie:
α to współczynnik siły oporu powietrza na jednostkę masy
g to przyśpieszenie ziemskie

Można zauważyć, że: $$ { \dot{x}(t)=V_x(t) ,\; \dot{y}(t)=V_y(t) } $$ $$ { \ddot{x}(t)=a_x(t) ,\; \ddot{y}(t)=a_y(t) } $$
Vx(0) i Vy(0) są równe: $$ { V_x(0)=v_0\cos(\Theta) } $$ $$ { V_x(0)=v_0\sin(\Theta) } $$
